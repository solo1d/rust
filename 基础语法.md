### if语句

```rust
fn main() {
    let a = if true {3} else {4};
    println!("a = {} ", a) // 输出 3
}
```

### 循环语句

- `loop、while、for`

loop 和 while(ture) 相同，无条件循环

```rust
loop { 循环内容 }

for i in 0..3
{ 循环3次，从0开始，不包括3 }
for i in 0..=3
{ 循环4次，从0开始，包括3 }

// 循环数组数据，拿元素的引用
let arrays = [2,4,6,8,10];
for i in arrays.iter(){  // i 是引用
    print!("arrays[i] = {} 、", i)
    // arrays[i] = 2 、arrays[i] = 4 、arrays[i] = 6 、arrays[i] = 8 、arrays[i] = 10
}
```

### 闭包

```rust
fn main(){
  let fx = |x| x * 2;  // 基础闭包
  let num = fx(2); // 调用闭包， num = 4
}
```

### match表达式

```rust
fn req_status() -> u32{
    200
}
fn main() {
    let status = req_status();
    match status {
        200 => println!("Source"),
        404 => println!("Not Found"),
        other =>{
            println!("Requset failed with code: {}" ,other);
        }
    };
}

```
