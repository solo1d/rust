## 断言
```rust
fn main() {
    let a = 1;
    let b = 2;
    assert!(a == b, "{} was not equal to {}", a, b); // 不为true 则输出后续错误信息，并停止进程
    assert_eq!(a,b, "{} and {} are not equal", a, b ); // 对比两个值，如果不相等，也是同样处理
    debug_assert!(a == b , "debug assert {} and {} are not equal", a, b ) // 在debug模式下才生效
}
```

## 单元测试
```rust
#main.rs
fn main() {
        print!("main!!!");    // 由于是跑下面的测试代码，而且还没有调用这里，所以它是不会执行的。
}

#[test]
fn basic_test(){
    assert!(true,"run test basci_test");
}
```
```bash
# 单个测试文件的编译测试命令
rustc --test main.rs     #会进行编译test内容，而且会生成一个main 的可执行文件， 直接执行即可。
# 项目
cargo test   #运行项目中全部的测试
```
